Case Study : Login Logout Authentication

The Aim of this project is to authenticate a user based on whether he/she os registered on our database or not and to verify the login credentials of the registered users.

Project Overview

The project consist of the front-end as well as back-end . In the front-end part Bootstrap framework is used for the designing part and in the back-end mongodb is used for store the user data and along with mongodb , node js is used to setup the server and to handle the routes for the smooth routing over the different pages.

Technology

Font-End Technolgies

Bootstrap

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains CSS- and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components

CSS

CSS(Cascading style sheet) is used for the designing of the elements and to beautify the content on the webpage by adding some color , text styles and transitions.

Back-End Technologies

Node Js

Node Js is used for non-blocking, event-driven servers, due to its single-threaded nature. It's used for traditional web sites and back-end API services, but was designed with real-time, push-based architectures in mind.

Mongo DB

MongoDB makes use of collections and documents. Documents consist of key-value pairs which are the basic unit of data in MongoDB.
